# OTel Learning Project

Iterative observability learning project — building understanding of OpenTelemetry, Prometheus, Grafana, Kubernetes, and CI/CD through hands-on implementation.

## Architecture Progression

1. **EC2 + Docker Compose** → Understand containers and telemetry data flow
2. **EKS migration** → Understand what orchestration solves
3. **Self-managed → Managed (AMP/AMG)** → Understand what you're paying for
4. **Simple pipeline → Production-grade** → Understand CI/CD maturity
